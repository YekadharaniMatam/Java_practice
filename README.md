# Java Practice Programs

This repository contains all the Java programs I have practiced during my training. 
Currently, it includes programs on basic Java and sorting algorithms. 
More programs on Object-Oriented Programming (OOP) and Data Structures & Algorithms (DSA) will be added soon.

## Skills and Concepts
- Java Basics: variables, loops, conditionals
- Sorting Algorithms
- Problem Solving and Practice
- (OOP and DSA programs coming soon)

## How to Run
1. Clone this repository to your local machine:
   git clone https://github.com/YekadharaniMatam/Java-Practice.git
2. Open the folder in your favorite IDE (IntelliJ, Eclipse, VS Code).
3. Compile and run the Java files using:
   javac FileName.java
   java FileName
